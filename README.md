# 🚀 OpenAI Codex 全方位教程

> 从零开始，一步步学会使用 Codex —— 你的 AI 编程搭档

---

## 📖 关于本教程

本教程面向**零基础用户**，用最通俗的语言、最详细的步骤、最丰富的配图，带你从安装 Codex 开始，逐步掌握全部功能。无论你是：
- 刚听说 Codex 想了解它是什么
- 已经用过 ChatGPT 网页版想升级工具
- 想系统学习 Codex 的所有能力

这本教程都适合你。
